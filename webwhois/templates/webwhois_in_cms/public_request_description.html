{% trans "Form to send the password request" %}

{% blocktrans trimmed %} This form sends the request for a password required to change the sponsoring registrar of an object in the registry (domains, contacts or name server set). Use the form in case you are not able to get this password directly directly from the current sponsoring registrar. {% endblocktrans %}

{% trans "Request for enhanced object security in CZ.NIC database." %}

{% blocktrans trimmed %} With this form, you can request to block the change of the sponsoring registrar for a domain, contact, or a name server set, or you can block the change of a domain, contact, or a name server set. {% endblocktrans %}

{% blocktrans trimmed %} Once the change of the sponsoring registrar is blocked, you cannot transfer the particular object to another registrar nor request the sending of a password needed for email changes in the central registry. {% endblocktrans %}

{% blocktrans trimmed %} Once data changes are blocked, you cannot make any change to the data related to the particular object in the central registry. If you wish to cancel the blocking options above, please continue to the next paragraph. {% endblocktrans %}

{% trans "Request to cancel the enhanced object security in CZ.NIC database." %}

{% blocktrans trimmed %} With this form, you can cancel the blocking of the change of the sponsoring registrar or cancel the blocking of any change of a domain, contact, or a name server set. {% endblocktrans %}
